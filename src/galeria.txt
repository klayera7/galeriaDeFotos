DOCUMENTAÇÃO DO PROJETO: GALERIA DE FOTOS (PEXELS API)
---------------------------------------------------------
Este arquivo contém o código-fonte completo para a aplicação de Galeria de Fotos, desenvolvida como parte de um desafio técnico Front-end. O projeto consiste em uma interface responsiva que consome a API da Pexels para buscar e exibir imagens de alta qualidade em tempo real.

RESUMO TÉCNICO:
1. Estrutura (HTML): Desenvolvida com semântica rigorosa (tags main, article, header) para garantir acessibilidade e SEO. Inclui feedback visual de carregamento (loader) e tratamento para estados vazios.

2. Estilização (CSS): Implementa um tema escuro (Dark Mode) com paleta de cores em tons de roxo, utilizando Variáveis CSS (:root) para fácil manutenção. O layout é responsivo via CSS Grid e Flexbox, adaptando-se a qualquer tamanho de tela.

3. Lógica (JavaScript):
   - Arquitetura: Código modularizado (ES Modules) separando a lógica da API (initApiKey) da execução principal.
   - Assincronismo: Utiliza async/await e Fetch API para chamadas de rede eficientes.
   - Funcionalidades: Busca dinâmica por termos, carregamento inicial de fotos "curated" e tratamento de erros (try/catch) para falhas de rede ou busca sem resultados.

INSTRUÇÕES PARA EXECUÇÃO:
Como o projeto utiliza Módulos ES, este código deve ser separado em seus respectivos arquivos (index.html, style.css, script.js e APY_KEY.js) e executado através de um servidor local (como o Live Server do VS Code) para evitar bloqueios de CORS. A chave da API já está configurada para testes.
